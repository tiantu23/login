垃圾分类 - 资源回收

做好垃圾分类回收，让资源循环再生，守护绿色地球家园

垃圾分类指南

掌握四大类垃圾分类标准，清晰区分不同垃圾类型，让垃圾分类变得简单易懂，助力绿色生活。

可回收物

指适宜回收利用和资源化利用的生活废弃物，主要包括：
✓ 纸类（报纸、纸箱、书本等）
✓ 塑料（饮料瓶、塑料盒、塑料袋等）
✓ 玻璃（玻璃瓶、玻璃杯、玻璃制品等）
✓ 金属（易拉罐、金属餐具、五金配件等）
✓ 纺织物（旧衣物、床单、窗帘等）

厨余垃圾

指易腐烂的、含有机质的生活垃圾，主要包括：
✓ 剩菜剩饭、火锅汤底、鱼骨鱼刺
✓ 瓜果皮核、蔬菜茎叶、腐坏食材
✓ 茶叶渣、咖啡渣、中药药渣
✓ 过期食品、宠物饲料、蛋壳等

有害垃圾

指含有对人体健康或环境有害的物质，主要包括：
✓ 电池（干电池、充电电池、纽扣电池）
✓ 灯管（荧光灯、节能灯、日光灯）
✓ 药品（过期药品、药瓶、药盒）
✓ 油漆、农药、化妆品、温度计等

其他垃圾

指除可回收物、厨余垃圾、有害垃圾外的生活垃圾，主要包括：
✓ 一次性餐具、塑料袋、保鲜膜
✓ 烟头、纸巾、尿不湿、卫生巾
✓ 破旧陶瓷、渣土、毛发、灰尘
✓ 污染的塑料袋、一次性口罩等

垃圾的危害科普

对人类健康

有害垃圾中的重金属、化学物质会通过空气、水源进入人体，引发中毒、癌症、神经系统损伤等疾病；厨余垃圾腐败产生的细菌和异味会导致呼吸道疾病和肠胃疾病。

对生态环境

塑料垃圾难以降解，会破坏土壤结构；有害垃圾渗漏会污染土壤和地下水；垃圾焚烧产生的二噁英等有毒气体造成大气污染，加剧温室效应。

对地球生态

全球每年产生超20亿吨垃圾，大量垃圾填埋占用土地资源，破坏地表植被；海洋垃圾形成“垃圾岛”，破坏海洋生态平衡，影响地球碳循环。

对水资源

垃圾渗滤液含有重金属和有毒有机物，渗入地下水或流入河流湖泊，导致水质恶化，破坏水生生态系统，威胁饮用水安全。

对动物生存

海洋动物误食塑料垃圾导致死亡；陆地动物被废弃渔网、塑料袋缠绕受伤；有害垃圾污染的水源和食物导致动物中毒、基因突变甚至灭绝。
